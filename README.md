### 📂 Main Projects

> All projects are private but here I highlight the main features.

#### 1️⃣ **BeMySun**  
Flutter app for **light and sound therapy**, managed by caregivers to assist patients.  
- Smart lamp control  
- Programmable light therapy sessions  
- Music and voice playback  

#### 2️⃣ **Desmo WebApp**  
Flutter PWA for **Desmotec gym instructors**, integrated with the backoffice.  
- Session history overview  
- PDF report generation  
- Multimedia content for each machine  

#### 3️⃣ **Vmini App**  
Cross-platform app for **exercise monitoring and group training**.  
- BLE device integration  
- Real-time performance feedback with custom graphs  
- Individual and group training configurations  

#### 4️⃣ **Import Your Card**  
Flutter app for **automated business card scanning**.  
- Extract contact information  
- Save contacts to device and CRM  
- Directly call saved contacts  

#### 5️⃣ **Digital Factory 5.0**  
NodeJS **Industry 5.0 solution** for machine communication and Logstash integration.  
- REST API for machine management  
- Multi-database support (MongoDB, PostgreSQL)  
- Windows service with PM2 and automation
